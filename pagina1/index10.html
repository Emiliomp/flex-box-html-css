JavaScript

Introduccion ¿Que es JavaScript?

JavaScript es un lenguaje de programación dinámico, versátil y orientado a objetos que se utiliza principalmente para desarrollar páginas web interactivas. Fue creado en 1995 por Brendan Eich y es uno de los lenguajes fundamentales del desarrollo web, junto con HTML y CSS.

¿Como funciona?

JavaScript es un lenguaje de programación que se ejecuta en el navegador web y permite que las páginas sean interactivas y dinámicas, manipulando el contenido, el diseño y la estructura del documento (DOM) en tiempo real. Funciona respondiendo a eventos del usuario, como clics o movimientos del ratón, y puede realizar tareas asíncronas, como cargar datos de un servidor sin recargar la página. Además, se integra con bibliotecas y frameworks para crear aplicaciones web complejas, mejorando la experiencia del usuario en la web.

Objetos en JavaScript

En JavaScript, un objeto es una estructura de datos que permite almacenar información en pares clave-valor, lo que lo convierte en una herramienta flexible para organizar y manipular datos de manera eficiente. Cada clave de un objeto es única y está asociada a un valor, que puede ser de cualquier tipo de dato, como números, cadenas, funciones o incluso otros objetos. Los objetos pueden representar entidades del mundo real, como una persona, un coche o un libro, al contener tanto sus características (propiedades) como su comportamiento (métodos).
Los objetos se dividen en 3 tipos:

  • Objetos Globales: Son aquellos que están disponibles en cualquier entorno de ejecución de JavaScript, ya sea en un navegador o en Node.js. Estos objetos están incorporados en el lenguaje y proporcionan funcionalidades básicas que facilitan la manipulación de datos y la ejecución de tareas comunes.
  • Objetos del navegador: Los objetos del navegador (también llamados Browser Object Model - BOM) son aquellos que permiten interactuar con el entorno del navegador. Estos objetos proporcionan acceso a funcionalidades que están directamente relacionadas con la ventana del navegador, la URL actual, las cookies, el historial de navegación, entre otros.
  • objetos del DOM: Los objetos del DOM están relacionados con el Document Object Model, que es la representación estructurada del documento HTML de una página web. El DOM permite a JavaScript acceder y manipular el contenido y la estructura de un documento de forma dinámica.

¿Para que se utiliza JS?

JavaScript se utiliza principalmente para agregar interactividad, dinamismo y funcionalidad avanzada a las páginas web. En el desarrollo frontend, permite manipular el DOM (Document Object Model) para actualizar el contenido de la página de manera dinámica sin necesidad de recargarla. También se usa para responder a eventos del usuario como clics, desplazamientos o pulsaciones de teclas, validar formularios, y crear animaciones o efectos
Además de su uso en el lado del cliente, JavaScript es utilizado en el backend gracias a tecnologías como Node.js, lo que permite desarrollar servidores y aplicaciones web completas. En este contexto, se usa para manejar solicitudes, gestionar bases de datos, crear APIs RESTful, manejar sesiones de usuario y realizar tareas en el servidor de manera eficiente.

Frameworks

Un framework es una estructura o plataforma de software predefinida que proporciona un conjunto de herramientas, bibliotecas y directrices para facilitar el desarrollo de aplicaciones. En lugar de empezar desde cero, los desarrolladores pueden aprovechar un framework para resolver problemas comunes y seguir buenas prácticas, lo que acelera el desarrollo y mejora la calidad del código.
Los frameworks establecen una arquitectura y un flujo de trabajo organizados, con el objetivo de simplificar tareas repetitivas y complejas, como la gestión de bases de datos, la manipulación de interfaces de usuario o la configuración de rutas en aplicaciones web. También suelen incluir plantillas, bibliotecas, y utilidades que ayudan en la construcción de funcionalidades específicas.
